# GSE320498 BALB/c Adriamycin Nephropathy Substrain Re-analysis — Code

This repository contains the **analysis code only** (R and Python scripts) used
to re-analyze GSE320498 (BALB/cAJcl vs BALB/cByJcl adriamycin nephropathy
glomerular RNA-seq) in response to peer review.

**Generated outputs (figures, tables, logs, supplementary packages) are not
tracked in this repository.** They are large, fully regenerable from the
scripts here plus the deposited GEO count matrices, and are archived
separately (Zenodo / local project storage) rather than version-controlled
alongside the code.

## Contents

- `scripts/` — the numbered primary pipeline (`00`–`20`), run in order,
  starting from the deposited GEO count matrices (`00_load_merge_qc.R`)
  through DE testing (DESeq2), GSEA (`fgsea`), pathway ORA (`ReactomePA`),
  and QC/sensitivity checks.
- `REVISION_PACKAGE_v2/scripts/` — later scripts (`21`–`37`) that build the
  final submission figures/tables (Figure 5–7, Supplementary Figures/Tables
  S1–S6) directly from the outputs of the scripts above — no new statistical
  models, pure re-visualization/repackaging unless a script's own header
  states otherwise.
- `genesets/KARAISKOS2018_PODOCYTE.gmt` — the one small external input file
  the scripts depend on (podocyte marker panel from Karaiskos et al. 2018,
  J Am Soc Nephrol 29:2060-2068); see `Karaiskos_geneset_provenance.md` for
  its full sourcing record.
- `histology/od_pipeline.py` — separate, independent pipeline: automated
  glomerular ROI detection and NPHS1 (nephrin) IHC DAB optical-density
  quantification from histology images (not part of the RNA-seq re-analysis
  above). See `histology/README.md` for its relation to Supplementary
  Figures S2/S5.
- Top-level `.md` files — the analysis log, methods rewrite proposal,
  reviewer-response material, and verification write-ups. See
  `README_analysis_log.md` first; `MANIFEST.md` is the original full-package
  index and still describes the `figures/`/`tables/`/`logs/` folders that
  exist in the full (non-code) archive but are not part of this repository.

## Reproducing the outputs

Run the scripts in `scripts/` in numeric order, then
`REVISION_PACKAGE_v2/scripts/` in numeric order, starting from the GSE320498
deposited count matrices. Each script's header comment states its inputs,
whether it fits a new statistical model or only re-visualizes an existing
one, and (where relevant) a consistency/verification check run against
canonical values.

## Data / full output archive

Figures, tables, logs, and the full submission-ready supplementary package
are not included here. They are regenerated by the scripts above and
maintained separately from this code repository.
